void pairwise_dist(double** X, double** D, int n, int dim) {
    for (int i = 0; i < n; ++i) {
        for (int j = 0; j < n; ++j) {
            double d = 0.0;
            for (int k = 0; k < dim; ++k) {
                double diff = X[i][k] - X[j][k];
                d += diff * diff;
            }
            D[i][j] = sqrt(d);
        }
    }
}

Why is Python Slow?

Interpreted

Translates bytecode at runtime, creating massive instruction routing overhead.

Dynamically Typed

Constant type-checking and method lookups bottleneck tight mathematical loops.

The GIL (Global Interpreter Lock)

Mutex prevents true multi-core execution for CPU-bound tasks within a single process.

High-Level Objects

Heavy C-structures fragment memory and destroy CPU cache locality.

void pairwise_dist(
    double** X, double** D,
    int n, int dim
) {
    for (int i = 0; i < n; ++i) {
        for (int j = 0; j < n; ++j) {
            double d = 0.0;
            for (int k = 0; k < dim; ++k) {
                double diff = X[i][k] - X[j][k];
                d += diff * diff;
            }
            D[i][j] = sqrt(d);
        }
    }
}
